# УБЕЖИЩЕ (Asylum Horror)

"Убежище" — это полноценный психологический хоррор от первого лица, созданный на чистом JavaScript с использованием библиотеки Three.js. Игра работает прямо в браузере и не требует установки сторонних движков.

## 🎮 Об игре
Вы заперты в жутком комплексе, где проводились запрещенные эксперименты. Ваша задача — выбраться, избегая встречи с существом, которое ориентируется на звук и движение. 

*   **Атмосфера:** Мрачные коридоры, мерцающий свет, давящий туман и пробирающий до костей саундтрек.
*   **ИИ Монстра:** Враг патрулирует карту, слышит ваши шаги и активно ищет вас, если заметит.
*   **Механики:** Система инвентаря, сбор предметов (ключи, батарейки, записки), использование фонарика и возможность прятаться в шкафах.
*   **Концовки:** В игре реализовано три различные концовки, зависящие от ваших действий и собранных улик.

## ⌨️ Управление
*   **W, A, S, D** — Передвижение
*   **Shift** — Бег (тратит выносливость)
*   **F** — Фонарик (берегите заряд батареи!)
*   **E** — Взаимодействие (открыть дверь, взять предмет, спрятаться)
*   **I / Tab** — Инвентарь
*   **ESC** — Выход из меню/игры

## 🛠 Технические особенности
*   **Движок:** Three.js (WebGL).
*   **Звук:** Программный синтез через Web Audio API.
*   **Графика:** Процедурная генерация текстур через HTML5 Canvas.
*   **Сохранения:** Прогресс сохраняется автоматически через `localStorage`.

## 🚀 Как запустить
### Локально
1. Скачайте все файлы проекта в одну папку.
2. Откройте `index.html` в любом современном браузере (Chrome, Edge, Firefox).
   * *Примечание: Для корректной работы захвата мыши используйте локальный сервер (например, Live Server в VS Code).*

### Через GitHub Pages (Онлайн)
1. Загрузите файлы в репозиторий GitHub.
2. Перейдите в **Settings -> Pages**.
3. В разделе **Build and deployment** выберите ветку `main`.
4. Через пару минут игра будет доступна по прямой ссылке, которой можно делиться с друзьями.
